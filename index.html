# 원본 주소 결과 정제된 도로명주소 우편번호
조건에 맞는 결과가 없습니다.

사용 방법

01
엑셀 업로드
주소 정보가 포함된 .xlsx 파일을 드래그 앤 드롭 또는 클릭으로 업로드합니다.
02
컬럼 선택
주소가 있는 컬럼을 선택합니다. 주소 관련 컬럼은 자동으로 감지됩니다.
03
일괄 정제
행정안전부 API로 각 주소를 검색해 표준 도로명주소와 우편번호로 변환합니다.
04
결과 다운로드
원본 주소·정제 주소·우편번호가 포함된 엑셀 파일을 즉시 저장합니다.
